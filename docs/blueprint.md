# **App Name**: VibrantChoice

## Core Features:

- Data File Input: Allow users to upload 'users.txt' and 'favourites.txt' files or directly paste their contents for analysis.
- Popular Color Analysis: A backend API route to process the input data, identify the most popular color based on user votes, and retrieve associated user information.
- Most Popular Color Display: Clearly present the name of the determined most popular color on the user interface.
- Voter List Display: Show a sortable and paginated list of full names (First Name Surname) of all users who voted for the most popular color.
- Popularity Statistics: Display key statistics, such as the total count of votes for the most popular color and the overall number of users participating in the poll.

## Style Guidelines:

- Light color scheme to enhance data readability. Primary accent color: A professional and trustworthy deep blue (#1A5DE6). Background color: A subtle, light blue-gray that offers a clean canvas (#F0F1F5). Secondary accent color: A contrasting, vibrant sky blue (#52C9E0) for highlights.
- Headline and body text font: 'Inter', a modern sans-serif typeface, chosen for its clarity, objectivity, and excellent readability for presenting data-heavy content.
- Implement a suite of clean, functional vector icons (e.g., upload, sort, search) to aid navigation and visual understanding without clutter.
- Employ a responsive grid-based layout ensuring clear segmentation of information and an optimal viewing experience across various devices. Emphasize whitespace for clarity and focus on the data.
- Incorporate subtle animations for state changes and data loading (e.g., when files are uploaded, results are processed, or table sorting occurs) to provide fluid user feedback.
